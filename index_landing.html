Summary: Latitude vs X

The purpose of this project was to analyze four distinct weather factors (temperature, cloudiness, humidity and wind speed) in search of a relationship between latitudinal proximity to the equator. The methodology applied was to collect weather data for over 500 randomly selected cities using the OpenWeatherMap API in order to create an operational dataset for analysis.


To facilitate analysis MatPlotLib was used to transform the data into visually representative charts, each of which can be viewed in greater detail by following the links provided along with the observations made and conclusions drawn. The source data is also available in this site for those interested in learning more about the measurement locations, factors and ranges used in the study.
